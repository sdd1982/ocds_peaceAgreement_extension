# Peace Agreement extension

## Descripción
Agregar el objeto peaceAgreement a Tender para indicar si el proceso de contratación está asociado al acuerdo de paz y bajo qué pilar específico.


## Example
``` json
{
	"tender":
	{
		"peaceAgreement":
		{
			"id":27,
			"number":7,
			"pillar":"TG.1. Capítulo de Género "
		}
	}
}
```
